Нужно осуществлять обход файловой системы и поиск заданного текста в файловой системе.

public class ParallelSearch {

    public List<String> search(String rootPath,
                               String text,
                               List<String> exts) {

    }

}

,где
rootPath - путь до папки откуда надо осуществлять поиск,
text - заданных текст,
exts - расширения файлов в которых нужно делать поиск.

Метод search() должен искать тест в файлах и возвращать адреса всех файлов
в которых был найден указанный текст.

Логика приложения.

1. Запустить код.
2. Внутри запустить несколько потоков.
3. Дождаться завершения поиска, и собрать результат в один список и вернуть его.

Напишите интеграционные тесты для того, чтобы убедиться в правильной работе
вашего решения.
