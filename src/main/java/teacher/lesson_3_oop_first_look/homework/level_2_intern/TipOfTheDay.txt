В языке программирования Java класс - это шаблон или формальное
описание, класс определяет свойства и методы, которые
будут присутствовать в объектах, созданных на основе этого класса.
Класс является основным строительным блоком программы на Java.

Давайте рассмотрим примеры классов в языке программирования Java:

Пример класса "Собака":
   Создадим класс "Собака", который определяет свойства и методы,
   присущие собаке. Этот класс может быть использован для создания
   объектов, представляющих различные собаки.

   ```java
   public class Dog {
       // Свойства класса
       String name;
       int age;

       // Методы класса
       public void bark() {
           System.out.println("Woof woof!");
       }

       public void run() {
           System.out.println(name + " is running!");
       }
   }
   ```

После определения класса мы можем создать объекты на основе этих классов:

```java
// Создание объекта класса "Собака"
Dog myDog = new Dog();
myDog.name = "Бобик";
myDog.age = 3;

// Вызов методов объекта
myDog.bark();  // Выводит: Woof woof!
myDog.run();  // Выводит: Бобик is running!
```

В приведенном примере "Собака" - это класс, который
определяет свойства и методы. Объекты, созданные на основе
этого класса, имеют свои собственные значения свойств
и могут вызывать методы класса для выполнения определенных операций.
