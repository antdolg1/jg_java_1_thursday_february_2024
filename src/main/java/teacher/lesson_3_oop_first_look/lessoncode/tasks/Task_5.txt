Создать в классе BankAccount методы:
- getOwner()
- getMoneyAmount()
для получения значения свойств объекта после его создания.
Воспользоваться созданными get() методами в классе BankAccountApp
для вывода значений свойств объекта на консоль.

Цели:
- научиться создавать get() методы;
- научиться использовать get() методы.
