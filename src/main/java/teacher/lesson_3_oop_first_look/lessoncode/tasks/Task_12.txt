Создать класс, который описывает ответ на вопрос анкеты.
У объектов этого класса должны быть следующие свойства:
- текст вопроса;
- ответ (в формате ДА/НЕТ).

* - вопросы подразумевают ответы только в формате ДА/НЕТ.

Цель:
- научиться создавать классы для представления
  сложных объектов из реального мира;
- познакомиться с типом данных boolean (true/false).
