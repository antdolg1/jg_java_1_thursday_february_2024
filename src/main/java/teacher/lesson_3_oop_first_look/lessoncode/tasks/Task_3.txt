Создать в классе BankAccount конструктор для задания
владельца банковского аккаунта.

Цели:
- познакомиться с понятием конструктор класса и его целью
  задания начальных значений для свойств создаваемого объекта;
- продемонстрировать как выглядит дефолтный конструктор класса;
- научиться создавать объекты (экземпляры класса) с параметрами.
