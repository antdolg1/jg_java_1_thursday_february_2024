Создать класс, который описывает страну.
У объектов этого класса должны быть следующие свойства:
- название страны (Латвия, Эстония и так далее);
- код страны (LV, EE, ...);
- население (количество жителей проживающих в стране).

Цель:
- научиться создавать классы для представления
  сложных объектов из реального мира.
