Создать классы, которые описывают семью.
Семья состоит из мамы, папы и двоих детей.

Объект мама:
- имя
- фамилия
- возраст в полных годах

Объект папа:
- имя
- фамилия
- возраст в полных годах

Объект ребёнок:
- имя
- фамилия
- возраст в полных годах.

Продемонстрируйте на примере как пользоваться созданными классами.
Создайте с помощью них две разные семьи.

Цель:
- научиться создавать классы для представления
  сложных объектов из реального мира.
