Создать в классе BankAccount методы:
- setOwner()
- setMoneyAmount()
для изменений значения свойств объекта после его создания.
Воспользоваться созданными set() методами в классе BankAccountApp
для изменения свойств объекта после его создания.

Цели:
- научиться создавать set() методы;
- научиться использовать set() методы для изменения значений
  свойств объекта после его создания.
