try-catch-finally

1. Блок try
Содержит код, который может вызвать исключение.
try {
    //код, который может вызвать исключение
}

2. Блок catch
Перехват и обработка исключений, которые могут возникнуть в блоке try
catch (ТипИсключения названиеПеременной) {
    //код для обработки исключения
}

3. Блок finally
Необязателен. Код, который должен быть выполнен в любом случае, независимо от того, возникло исключение или нет.
finally {
    //код для выполнения
}
